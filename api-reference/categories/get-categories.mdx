---
title: 'List categories'
description: 'Retrieve all categories available in your personal finance account.'
---

Returns a list of all categories. Use these categories when creating or filtering transactions.

## Endpoint

```
GET /api/categories
```

## Parameters

This endpoint does not accept any query parameters.

## Response

<ResponseField name="id" type="string">
  Unique identifier for the category, assigned by the server.
</ResponseField>

<ResponseField name="name" type="string" required>
  Display name of the category.
</ResponseField>

## Examples

<CodeGroup>

```bash cURL
curl --request GET \
  --url "${VITE_API_URL}/api/categories"
```

```typescript TypeScript
import { getCategories } from './src/api/transactions';

const categories = await getCategories();
console.log(categories);
// [
//   { id: "1", name: "Food" },
//   { id: "2", name: "Transport" }
// ]
```

</CodeGroup>
