## What is Personal Finance Web?

Personal Finance Web is a full-featured personal finance management application built with React, TypeScript, and Tailwind CSS. It connects to a REST API backend and gives you a clear picture of your financial health through an interactive dashboard, detailed transaction logs, and customizable spending categories.

## Key features

  <Card title="Financial dashboard" icon="gauge">

  </Card>
  <Card title="Interactive charts" icon="chart-pie">
    Bar, pie, and donut charts powered by Apache ECharts showing spending by category and over time.
  </Card>
  <Card title="Transaction management" icon="list-check">

  </Card>
  <Card title="Category management" icon="tags">
    Organize spending with custom categories that can be created, renamed, and deleted.
  </Card>
  <Card title="Advanced filtering" icon="filter">

## Tech stack

| Layer | Technology |
|---|---|
| Framework | React 19 + TypeScript |
| Routing | React Router v7 |
| Styling | Tailwind CSS v4 |
| Charts | Apache ECharts via echarts-for-react |
| Build tool | Vite |
| HTTP | Native Fetch API |
| Currency | Brazilian Real (BRL) via `Intl.NumberFormat` |

## Architecture overview

The frontend is a single-page application that communicates with a separate REST API backend. All state is managed through React Context providers, with lazy-loaded page components for optimal performance.

<Info>
  The backend API URL is configured via the `VITE_API_URL` environment variable. You must have the backend running before using most features.
</Info>
